# Özkan Doğu — Portfolio

Personal portfolio for **Özkan Doğu**, Senior Software Engineer.

Built with Next.js 15, TypeScript, Tailwind CSS v4, and framer-motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** framer-motion, @react-spring/web
- **Fonts:** Inter, JetBrains Mono

## Project Structure

```
src/app/
├── sections/          # Page sections (Hero, About, FeaturedProject, etc.)
├── components/        # Reusable animation and UI components
├── about/             # /about page
├── contact/           # /contact page
└── my-projects/       # Project detail pages
    ├── seaborn-erp/
    ├── asendia-logistics/
    └── headless-commerce/
```

## Deploy

Deploy on [Vercel](https://vercel.com) for best Next.js compatibility.
